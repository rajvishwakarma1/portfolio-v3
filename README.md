## Logo Guidelines

For cleanest display, upload logos at:

| Display Size | Recommended Upload | Format |
|:-------------|:-------------------|:-------|
| 48×48px | **128×128px** | PNG/JPG |

- Square format (1:1 aspect ratio)
- 128×128px minimum for retina displays
- PNG for transparency, JPG for photos
